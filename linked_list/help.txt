[*] help :
        Type `help` to list all commands.

[*] commands:
        - lss    :   Prints all Lists available.
        - ls    :   Prints List from memory, Returns Count.
        - ld <list> :   Loads List into memory.
        - uld <list> :   Unloads List from memory.
        - psh <item> :   Pushes item to last into List in memory.
        - ins <pos|item> :   Inserts item to given position or first into List in memory.
        - pop <item> :   Removes last item from List in memory.
        - rm <pos|item> :   Removes item by position or item from List in memory.
        - new <name> :   Creates new List in memory.
